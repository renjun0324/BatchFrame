## 🚀 项目简介

**批量边框助手** 是一款基于微信小程序开发的图片加边框工具。  
它可以快速为照片添加黑边、白边、双层相框等效果，并支持比例调整、批量处理和保存到相册。  

项目使用微信小程序官方框架构建，界面简洁，操作直观，适合摄影爱好者与设计师使用。

最适合发小红书。

---

## 🧩 功能特性

- ✅ **多种边框样式**：支持黑边、白边、胶片风格边框等  
- ✅ **实时预览**：边框厚度、比例即时可视化  
- ✅ **比例选择**：支持宽屏、竖屏、自定义比例  
- ✅ **批量处理**：一次导入多张照片并自动生成  
- ✅ **图片导出**：一键保存到相册
